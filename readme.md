Simple Website Template
========================

####A simple file structure created to produce a quick sinatra application for site hosting

#####Technologies Used:

  - Sinatra
  - Sinatra-partial
  - ERB
  - HTML
  - JQuery
  - CSS Normalize

How to Use:
------------

Clone the Repository:
```shell
$ git clone https://github.com/Scully87/Sinatra-Template.git
```

Open the Directory:
```shell
$ cd Sinatra-Template
```
Install the files:
```shell
$ bundle install
```

Open from terminal:
```shell
$ rackup
```
