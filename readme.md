Simple Website Template
========================

####A simple file structure created to produce a quick sinatra application for site building

#####Technologies Used:

  - Sinatra
  - Sinatra-partial
  - ERB
  - HTML
  - JQuery
  - CSS Reset
  - CSS Normalize
  - RSpec for Testing

How to Use:
------------

Clone the Repository:
```shell
$ git clone git@github.com:Scully87/Sinatra-Template.git
```

Open the Directory:
```shell
$ cd Sinatra-Template
```
Install the files:
```shell
$ bundle install
```

Open from terminal:
```shell
$ rackup
```

Url for Browser:
```
  http://localhost:9292/
```
